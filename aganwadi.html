Overview

What is Anganwadi Inventory Management?

This project aims to streamline and digitize the stock tracking system of Anganwadi centers across regions. It provides centralized monitoring and supply chain visibility from Central Head to local Anganwadi centers and suppliers.

User Roles

  • Head: Manages overall operations and inventory tracking.
  • Central Head: Regional authority overseeing multiple districts.
  • Anganwadi: Local center that receives stock and submits requirements.
  • Supplier: Delivers stock and updates delivery status.

Key Features

  • Real-time inventory tracking across all centers
  • Stock alerts and shortage notifications
  • Supplier assignment and delivery tracking
  • Role-based dashboard access (Admin, Anganwadi, Supplier, etc.)
  • Secure login with data validation

Current Status

  • UI is under development using HTML, CSS, Bootstrap
  • Backend logic with Python and Firebase integration
  • Real-time data flow using JSON format
  • Admin panel and user dashboard in progress
