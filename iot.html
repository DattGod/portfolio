Overview

What is IOT Farming System?

The IOT Farming System is a smart agriculture interface designed to help farmers monitor and manage field conditions like soil moisture, temperature, humidity, and more through a user-friendly dashboard. The design focuses on simplicity and clarity while delivering live data visualization and remote control options.

My Role

  • Planned and designed the complete UI/UX workflow.
  • Created wireframes, low/high fidelity prototypes.
  • Focused on responsive layouts for web/mobile view.
  • Used minimalistic colors, icons, and charts for data clarity.

Key Features in UI Design

  • Dashboard with real-time sensor readings
  • Map-based field view for IOT device location
  • Control panel for irrigation, lighting, and devices
  • Data chart for moisture and temperature logs
  • Alerts and automation configuration

Project Impact

  • Improved user experience for remote smart farming
  • Designed for practical usability by farmers
  • Supports sustainability through smart irrigation planning
