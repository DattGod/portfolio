Overview

What is WEBGEN?

WEBGEN is an AI-powered tool built to help developers generate HTML, CSS, and JavaScript code dynamically using natural language prompts. It leverages Hugging Face APIs and local LLMs such as Ollama for low-latency, privacy-friendly inference.

My Role

  • Designed and built the entire UI/UX and backend functionality.
  • Integrated Hugging Face transformer models for prompt-to-code logic.
  • Experimenting with local LLMs like Ollama for offline capability.

Key Features

  • Generate HTML/CSS/JS snippets from plain English
  • Copy & preview code instantly
  • Option to use local models (Ollama) for secure dev use
  • Clean, developer-friendly UI

Future Scope

  • Improve accuracy using fine-tuned Hugging Face models
  • Integrate real-time code validation and testing
  • Expand to include backend frameworks (Node.js, Django)
